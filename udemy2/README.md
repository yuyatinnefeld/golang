# Fintech Application with Golang

# setup 
```bash

cd udemy2
# create project
mkdir trading && cd trading

# create module
export GO111MODULE="on"
mkdir mymodule && cd mymodule
go mod init mymodule

# create package
mdkri config && nano config/config.go

# update packages
go mod tidy


# run the main.go
cd mymodule
go run main.go
```
